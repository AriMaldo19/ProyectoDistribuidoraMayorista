# Proyecto SQL Server + Python  
## Análisis de una Distribuidora Mayorista de Alimentos (2023–2025)

Proyecto de análisis de datos orientado a una distribuidora mayorista simulada, utilizando SQL Server para modelado relacional y consultas analíticas, y Python (Matplotlib) para visualización de insights clave.

---

## 🎯 Objetivo

Analizar el desempeño comercial de una distribuidora mayorista, evaluando ventas, rentabilidad, comportamiento de clientes y evolución temporal del negocio.

El proyecto se centra en la correcta formulación de preguntas de negocio, el uso eficiente de SQL para el análisis de datos y la representación visual de los principales hallazgos.

---

## 🛠 Tecnologías utilizadas

- SQL Server
- Python
- Matplotlib
- Git & GitHub

## 📊 Análisis realizados

### 🔹 Análisis básicos

- Volumen promedio de unidades por venta (validación del modelo mayorista)
- Evolución anual de ventas confirmadas vs canceladas
- Análisis del ticket promedio por año
- Estacionalidad de ventas (hemisferio sur)
- Control de márgenes unitarios

### 🔹 Análisis avanzados

- Evolución mensual de ventas y ganancias (2023–2025)
- Ranking de clientes por facturación y rentabilidad
- Rentabilidad por tipo de comercio
- Comparación interanual de ventas (Year-over-Year)
- Segmentación de clientes según frecuencia de compra
- Evaluación histórica de productos activos e inactivos

---

## 📈 Visualizaciones

Se generaron visualizaciones clave utilizando Matplotlib para representar los insights más relevantes del análisis, priorizando claridad y enfoque por sobre cantidad.

Principales gráficos desarrollados:

- Evolución mensual de ventas vs ganancias  
- Comparación interanual de ventas  
- Top 10 clientes por facturación  
- Rentabilidad por tipo de comercio  
- Volumen promedio de unidades por venta
